Quickly configure a fresh WordPress install — including full Elementor setup tasks.

== Description ==

Fresh Install Assistant gives you a single page to run all the tedious tasks
you do on every new WordPress site:

**Essential Cleanup**
* Remove "Hello World!" post
* Remove "Sample Page"
* Rename "Uncategorized" → "Blog"
* Set permalink structure to /%postname%/
* Remove Hello Dolly & Akismet plugins
* Disable comments & pingbacks sitewide
* Hide noisy dashboard widgets

**Initial Site Setup**
* Set site title & tagline
* Create starter pages (Home, Services, About, Contact, Blog)
* Set Home as front page, Blog as posts page
* Create & assign main navigation menu

**Elementor Setup** ⭐ NEW
* Install & Activate Elementor (if not present)
* Disable default Elementor colors & fonts (use theme values)
* Enable Flexbox Containers experiment
* Enable Grid Container experiment
* Enable Elementor for Pages & Posts
* Load Font Awesome 5 support
* Set default content width
* Disable default image lightbox
* Set CSS print method to Internal Embedding

**Theme Installation**
* Optionally install: Astra, GeneratePress, OceanWP, Blocksy, Neve,
  Hello Elementor, or Kadence

== Installation ==

1. Upload the `fresh-install-assistant` folder to `/wp-content/plugins/`
2. Activate via **Plugins → Installed Plugins**
3. Go to **Fresh Install (FIA)** in the left admin menu
4. Select your tasks and click **Run Setup**
